# Qwen2.5-3B LoRA - Raw Text Fine-Tuning

Fine-tuning Qwen2.5-3B-Instruct using LoRA + QLoRA on raw cybersecurity text.

## Features

- Sliding Window Dataset
- QLoRA (4-bit)
- Manual PyTorch Training Loop
- PEFT LoRA
- HuggingFace Transformers
- Custom Dataset

## Dataset

Raw cybersecurity notes, cheatsheets and documentation.

## Model

Qwen/Qwen2.5-3B-Instruct
